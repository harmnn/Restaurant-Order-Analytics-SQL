# Food Delivery SQL Analytics

## Project Overview
This project analyzes a food delivery business dataset using SQL.  
The goal is to extract business insights related to customer behavior, restaurant performance, order trends, and revenue generation.

---

## Business Problems Solved

This project answers questions such as:

- Who are the top spending customers?
- Which restaurants generate the highest revenue?
- What are the most frequently ordered menu items?
- What is the cancellation rate?
- Which cities contribute the most revenue?
- Who are repeat customers?

---

## Dataset

The project uses 5 relational tables:

1. orders_medium
2. order_items
3. customers_medium
4. menu_items
5. restaurants

---

## Skills Demonstrated

This project demonstrates:

- Joins
- Aggregations
- Group By + Having
- Subqueries
- Window Functions
- Ranking
